DVPP对外接口

DVPP对外接口

DVPP除VDEC模块对外提供三个接口，如下：

  • CreateDvppApi：负责创建DVPP API（Application Programming Interface）实例，DVPP API实例类似于打开驱动后返回的文件描述符。
  • DvppCtl：控制DVPP执行器进行视频编解码（主要调用各个模块暴露的Process接口函数，模块主要包括VPC（Vision Pre-processing Core）、VENC、JPEGE、JPEGE、PNGD等，其含义见缩略语和术语一览）。
  • DestroyDvppApi：为销毁DVPP API。

VDEC模块由于当前内部不是单例模式，所以其对外提供C接口函数与其他不同。接口使用请参考VDEC。
