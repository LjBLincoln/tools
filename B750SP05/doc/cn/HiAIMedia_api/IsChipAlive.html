IsChipAlive

IsChipAlive

查询Atlas DK开发者板上与Ascend 310对接的Hi3559芯片的当前连接状态是否正常。

函数格式

int IsChipAlive(char* chipName)

参数说明

参数

说明

取值范围

chipName

与Ascend 310对接的芯片名称。目前只支持hi3559

“3559”

返回值

返回的错误码请参见错误码示例中的“错误码”列。

错误码示例

真值

错误码

错误码描述

0

LIBMEDIA_STATUS_FAILED

与3559对接失败

1

LIBMEDIA_STATUS_OK

与3559对接正常
