工程导入

工程导入

  1. 在Mind Studio新建工程界面选择“Mind Engine Project > Mind Project“工程类型。

    • “Mind Type“选择“CUSTOM“。
    • “Target“选择“ASIC“（当前样例只支持在PCIE/EVB板上执行）。
    • “Source Code From“选择“Local (WebServer)“。
    • “Local File Path“中选择“$HOME/tools/che/ddk > ddk > sample > detection_ssd_asic“。

    单击“Create“完成样例工程导入。
