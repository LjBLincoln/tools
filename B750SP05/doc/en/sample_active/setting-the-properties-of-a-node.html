Setting the Properties of a Node

You can select a node to set its properties required for node running

on the property tab in the right pane, as shown in Figure 1.

Figure 1 Setting node properties

Different types of nodes have different properties, as described in Table 1.

From-Upper

point_x

Horizontal coordinate of the crop start. The value must be an even number.

point_y

Vertical coordinate of the crop start. The value must be an even number.

crop_width

Crop width. The value must be an even number within [1/32, 4].

crop_height

Crop height. The value must be an even number within [1/32, 4].

Resize

(resize enable)

resize_width

Width after resize. The value is a positive number up to 4096.

resizie_height

Mean Value (image pre-processing average)

mean_of_B

mean_of_R

Channel R for image pre-processing. The mean is the same as that in the training. The maximum value is 255.

mean_of_G

Channel G for image pre-processing. The mean is the same as that in the training. The maximum value is 255.

Scale

scale_value

Scaling coefficient. The maximum value is 255.
