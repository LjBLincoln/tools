（Extended） Engine Orchestration Without Preprocessing

（Extended） Engine Orchestration Without Preprocessing

Prerequisites

  • The model file and weight file complying with the Caffe framework are ready and imported to Caffe Models.

    NOTE:
    For details, see 5.3.2 "Adding a Caffe Model Component" in the Mind Studio Basic Operations.

  • You are familiar with the function described in Operating Procedure.

Table 1 describes the engine orchestration process and precautions for the open-source Caffe model without pre-processing.

Table 1 Engine orchestration process for the open-source Caffe model without pre-processing.

Orchestration

Emulation Environment

Creating a Mind project
  • Mind Type: Only DEFAULT is supported.
  • Target: Only Local (simulation environment) is supported.

Editing the network structure
  • Datasets: Only BGR images are supported. That is, only raw datasets are supported.
  • Model: Only a new component under Caffe Models is supported. The new component in this example is resnet18_caffe.
  • Inference engine: Only CaffeInferenceEngine is supported.

Compiling and running

For details, see Compiling and Running.

Viewing the running result

For details, see Viewing the Running Result.
