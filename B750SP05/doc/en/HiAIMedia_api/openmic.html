OpenMIC

OpenMIC

Opens the microphone on the Atlas DK developer board. A micro-electro-mechanical systems (MEMS) microphone is embedded in the board. For details, see the product manual. If you need to use the recording function, place the sound source close to the MEMS microphone as possible before start recording.

Syntax

int OpenMIC()
