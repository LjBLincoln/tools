IsChipAlive

IsChipAlive

Checks whether Hi3559 is properly connected with Ascend 310 on the Atlas DK developer board.

Syntax

int IsChipAlive(char* chipName)

Parameter Description

Parameter

Description

Value Range

chipName

Name of the chip interconnected with Ascend 310 Currently, only Hi3559 is supported.

The value is 3559.

Return Value

For details about the returned error codes, see the Error Code column in Error Codes.

Error Codes

True Value

Error Code

Description

0

LIBMEDIA_STATUS_FAILED

The interconnection with Hi3559 fails.

1

LIBMEDIA_STATUS_OK

The interconnection with Hi3559 is normal.
