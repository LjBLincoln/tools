QueryCameraIds

QueryCameraIds

Queries the available camera channel IDs on the Atlas DK developer board. Currently, the developer board supports a maximum of two cameras at the same time.

Syntax

uint32_t QueryCameraIds(int* cameraIds, uint32_t *count)

Parameter Description

Parameter

Description

Value Range

cameraIds

Array for storing the query result

An array with the size of at least 2*sizeof(int) should be provided to save the returned result.

NOTE:

If the size of the user-provided array exceeds the actual number of cameras, the function sets the invalid ID to –1. The valid ID is 0 or 1.

count

Number of IDs that can be contained in the cameraIds array space

[1,2]

Return Value

For details about the returned error codes, see the Error Code column in Error Codes.

Error Codes

True Value

Error Code

Description

0

LIBMEDIA_STATUS_FAILED

The query fails.

1

LIBMEDIA_STATUS_OK

The query is successful.
