Creating/Deleting a Project

Creating/Deleting a Project

  • Creating a project:

    In the Mind Studio main menu, choose File > New > New Project to create a Mind project for example. In the dialog box that is displayed, click Mind Project under Mind Engine Project, enter the project name, and click Create to create a project, as shown in Figure 1. Table 1 describes the parameters on the page.

    Figure 1 New Project dialog box

    NOTE:
    A newly created project is saved in the projects folder in the root directory of Mind Studio, and the directory cannot be changed.

    Table 1 Parameter description

    Parameter

    Parameter Description

    Name

    Set the project name. Set this parameter as required.

    The name must be a character string without spaces or Chinese characters. A space is automatically filled with a hyphen (-).

    Mind Type

    Select a project type from the drop-down list box.

    • DEFAULT: A canvas is generated as the project is created. You can orchestrate the project by dragging.
    • CUSTOM: A custom project is created without generating a canvas.

    Target

    Select a running environment from the drop-down list box.

    • Local: simulation environment, which is used only for the Caffe inference engine
    • ASIC: Evaluation Board (EVB) or PCIe board connected
    • Atlas DK: developer board connected
    NOTE:

    If a local simulation project is created, model components under Caffe Models in Model must be used. For details about how to add Caffe model components, see Adding a Caffe Model Component. The pre-processing node must be ImagePreProcessPillow and the inference engine must be CaffeInferenceEngine.

    Source Code

    Select a code source from the drop-down list box.

    • Empty: The project is not imported externally.
    • Local(Web Client): The source file is imported from the local Windows PC. The text box for uploading the source files displayed.
    • Local(Web Server): The source file is imported from the backend server. The text box for entering the code path of the Mind Studio server is displayed.

  • Deleting a project:

    Right-click the root directory of the project and choose Delete from the shortcut menu.
