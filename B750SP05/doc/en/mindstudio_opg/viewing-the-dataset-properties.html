Viewing the Dataset Properties

Viewing the Dataset Properties

The property configurations of a service node are the parameters required for running the node.

Select a dataset and set its properties on the property tab page in the right pane, as shown in Figure 1 and Figure 2.

Figure 1 Setting the properties of the built-in COCO100 dataset

Figure 2 Setting the properties of the custom raw dataset

Table 1 describes the node property parameters.

Table 1 Description of the node property parameters

Property Parameter

Description

Name

Dataset name, set in the Import Dataset dialog box

Path

Dataset path on Mind Studio, set in the Import Dataset dialog box

Data Type

Data type, set in the Import Dataset dialog box

Include YUV420SP

Whether a YUV420SP image is included in the dataset, set in the Import Dataset dialog box

Height

Image height, available only when Include YUV420SP is true or Data Type is Raw and set in the Import Dataset dialog box. The value is an integer.

Width

Image width, available only when Include YUV420SP is true or Data Type is Raw and set in the Import Dataset dialog box. The value is an integer.

Mean of B

Mean value of the B channel. This parameter is available only Data Type is Raw and set in the Import Dataset dialog box. The value is a floating-point number. A maximum of four decimal places are supported.

Mean of G

Mean value of the G channel. This parameter is available only Data Type is Raw and set in the Import Dataset dialog box. The value is a floating-point number. A maximum of four decimal places are supported.

Mean of R

Mean value of the R channel. This parameter is available only Data Type is Raw and set in the Import Dataset dialog box. The value is a floating-point number. A maximum of four decimal places are supported.

Batch

Number of images processed at a time

Run Mode

Running mode

  • All: Processes all the images.
  • Specify: Processes selected images.
  • Random: Processes Random Number images.

Random Number

This parameter is available only when Run Mode is set to Random. The value range is [1, Image count in the dataset). If the image count in the dataset is 1, this parameter can only be set to 1.
