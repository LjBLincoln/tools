Deleting a Custom Dataset

Deleting a Custom Dataset

  1. Right-click a dataset under My Datasets and choose Delete from the shortcut menu, as shown in Figure 1.

    Figure 1 Deleting a Custom Dataset

  2. On the Confirmation dialog box that is displayed, click Yes, as shown in Figure 2.

    Figure 2 Confirmation dialog box

  3. When the dialog box shown in Figure 3 is displayed, the dataset is successfully deleted.

    Figure 3 Successful deletion dialog box

  4. Click ok.

  5. In the Datasets Explorer, expand my-datasets, and click to refresh the directory. The directory of the dataset images is gone, as shown in Figure 4.

    Figure 4 Checking Datasets Explorer
