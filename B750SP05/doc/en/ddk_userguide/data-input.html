Parameter

Description

BGR test data to be classified

/test_data/data/faster_rcnn.bin in the project directory (180 x 300)

Faster-RCNN (vgg16) model

Model name: FasterRCNN-vgg16.davinci.om

Conversion method:

Place the faster_rcnn_vgg16 Caffe model file in the test_data/model_gen directory and rename the file FasterRCNN_VGG16_180x300_do.caffemodel.

Go to the test_data/model_gen directory and run the following commands:

1. su root

2. export LD_LIBRARY_PATH=/<Installation directory>/tools/che/ddk/ddk/uihost/lib

3. /<Installation directory>/tools/che/ddk/ddk/uihost/bin/omg --model=./FasterRCNN_VGG16_180x300_do.prototxt --weight=./FasterRCNN_VGG16_180x300_do.caffemodel --framework=0 --aipp_conf=aipp_nv12_img.cfg --output=./FasterRCNN-vgg16.davinci --op_name_map=./model_face.txt

FasterRCNN_VGG16_180x300_do.prototxt is the network model file, aipp_conf is the AIPP configuration file, and op_name_map is the operator mapping configuration file (specified when the DetectionOutput network is included). The three files can be obtained from the test_data/model_gen directory in the project directory.

For details about the parameters, see the Mind Studio Basic Operations.

Place the converted FasterRCNN-vgg16.davinci.om file in the test directory (the file is in the same directory as the executable my_vgg16 file).

Note: You can also convert the model by using the Mind Studio GUI. For details, see Mind Studio Basic Operations.

Graph configuration file

/test_data/config/sample.prototxt in the project directory
