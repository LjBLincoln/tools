Sample Data GeneratIon

Perform the following steps to manually execute the data generation script in the sample project on the Mind Studio server:

  1. Go to the path of the new project.

    For example, if the name of the new sample project is tvm-pro, run the following command:

    cd /projects/tvm-pro

  2. Run the DataGen.py script to generate sample data:

    python DataGen.py

    Eight data files as listed in Table 1 are generated. The files starting with sign are the input data and verification data for operators in test_tvm_sign.py. The files starting with reduction are the data for the subtraction operations.

    Table 1 Description of generated data files

    Data File

    Description

    reduction_input_2_3_4_sum_axis_1.data

    Input data file of the reduction operator in binary format

    reduction_input_2_3_4_sum_axis_1.txt

    Binary file of the reduction operator in .txt format for users to easily view the results

    reduction_output_2_3_4_sum_axis_1.data

    Output data verification file of the reduction operator in binary format, which is used to verify whether the output is correct after the operator runs

    reduction_output_2_3_4_sum_axis_1.txt

    Output data file of the reduction operator, which is visible to users and has the same functions as reduction_output_2_3_4_sum_axis_1.data.

    sign_input_2_4_cce.data

    Input data file of the sign operator in binary format

    sign_input_2_4_cce.txt

    Binary file of the sign operator in .txt for users to easily view the results

    sign_output_2_4_cce.data

    Output data verification file of the sign operator in binary format, which is used to verify whether the output is correct after the operator runs

    sign_output_2_4_cce.txt

    Output data file of the sign operator, which is visible to users and has the same functions as sign_output_2_4_cce.data

    The preceding data dimensions are all (2, 4), that is, in two rows and four columns.
