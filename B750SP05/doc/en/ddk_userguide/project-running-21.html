Project Running

Project Running

EVB/PCIe card: Place the generated executable file DetectionSSD and libDetectionSSDEngine.so in the same directory, create the test_data folder, create the ssd folder in the test_data folder, create the data, mode, and config folders in the ssd directory, place the src_test.bin file in the data folder, graph_sample.prototxt in the config folder, SSD-model.om in the mode folder, and run the DetectionSSD script. After the operation is successful, the 4-byte result_data_num_file file is generated, indicating the number of detected boxes, and the 5600-byte result_data_bbox_file file, indicating the details about the boxes.
