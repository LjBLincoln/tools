Project Import

Project Import

  1. In the New Project dialog box, choose Mind Engine Project > Mind Project.

    • Set Mind Type to CUSTOM.
    • Set Target to ASIC. (The current sample can be executed only on a PCIe card or EVB.)
    • Set Source Code From to Local (WebServer).
    • Set Local File Path to $HOME/tools/che/ddk > ddk > sample > classify_net_asic.

    Click Create to import the sample project.
