Project Import

Project Import

  1. In the New Project dialog box of Mind Studio, choose Operator Project > Tensor Engine Project, select Local (WebServer) from the Source Code From down list box, and set Local File Path to $HOME/tools/che/ddk > ddk > sample > tvm.

  2. Click Next. The model configuration dialog box is displayed, as shown in Figure 1.

    Figure 1 Network model configuration

    Set Model File to a model file. Select the check box of the first layer in the Custom Layer area.

    NOTE:
    You can import any model file converted by any tool so that custom operators can use its framework.

  3. Click Create. A Tensor Engine project is created by importing the sample project, as shown in Figure 2.

    Figure 2 Sample TVM project

    Table 1 describes the code directory structure.

    Table 1 Code directory structure

    Directory

    Description

    BatchNorm_plugin

    Code directory for the custom operator plug-ins

    This sample implements only operators. The implementation of the operator plug-in is not involved.

    operator

    Directory for storing the implementation code of an operator

    proto

    proto files of the built-in operators of the tool

    DataGen.py

    Script for generating sample data

    test_tvm_reduction.py

    Implementation sample of the reduction operator

    test_tvm_sign.py

    Implementation sample of the sign operator
