Changing the Password

Changing the Password

Before using the Atlas 200 DK developer board, you need to change the initial user password.

Changing the Password of a Common User

HwHiAiUser is the default user created during SD card preparation on Mind Studio. The default password is Mind@123. After the Atlas 200 DK developer board is successfully connected to Mind Studio, you need to change the initial password of the HwHiAiUser user.

  1. Log in to the Atlas 200 DK developer board as the HwHiAiUser user in SSH mode on the Mind Studio server.

    NOTE:
    The default login password of the HwHiAiUser user is Mind@123.

  2. Run the passwd command to change the password of the HwHiAiUser user.

    See Figure 1.

    Figure 1 Changing the password of the HwHiAiUser user

Modifying the Password of the Root User

  1. Log in to the Atlas 200 DK developer board as the HwHiAiUser user in SSH mode on the Mind Studio server.

    NOTE:
    The default login password of the HwHiAiUser user is Mind@123.

  2. Run the following command to switch to the root user:

    su - root

    NOTE:
    The default login password of the root user is Mind@123.

  3. Run the passwd command to change the password of the root user, as shown in Figure 2.

    Figure 2 Changing the password of the root user
