Environment Preparation

Environment Preparation

Hardware Environment

  • Prepare a 16 GB or larger SD card.

    NOTE:
    The card will be formatted. Back up the data in advance.

  • If a card reader is available:

    1. Insert the SD card into the card reader and connect the card reader to the USB port of the PC where Mind Studio is installed.
    2. Run the fdisk -l command on the host where Mind Studio is located to check the card reader, for example, /dev/sda.
    3. Perform SD Card. Select the device corresponding to the card reader.

  • If no card reader is available:

    1. Remove the upper case by referring to （Optional） Removing the Upper Case.

    2. Place the jumper over the following two pins on the developer board as shown in Figure 1.

      NOTE:
      Perform this operation when the developer board is powered off. For details about the power-off requirements, see Powering off the Atlas 200 DK Developer Board.

      Figure 1 Pin map of the developer board

    3. Insert the SD card into the SD card slot of the developer board, and connect the developer board to the USB port of the PC where Mind Studio is installed.

    4. Power on the developer board. For details, see Powering on the Atlas 200 DK Developer Board.

    5. Perform SD Card. Select the device corresponding to the USB device.

      Run the fdisk -l command on the host where Mind Studio is located to check the USB device, for example, /dev/sda.
