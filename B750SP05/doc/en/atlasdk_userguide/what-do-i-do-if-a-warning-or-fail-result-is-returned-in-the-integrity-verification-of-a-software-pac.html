What Do I Do If a WARNING or FAIL Result Is Returned in the Integrity Verification of a Software Package?

If a WARNING or FAIL result is returned in the integrity verification of a software package, the verification fails. Rectify the fault by referring to the handling suggestions described in Table 1.

Table 1 Verification result examples

Verification Result Description

Displayed Information

PASS

N/A

FAIL

Download the target file again.
