What Do I Do If a Log File Fails to Be Uploaded to Mind Studio Due to the Incorrect Format?

Symptom

The format of the log data block is incorrect. As a result, the log data block cannot be uploaded to the Mind Studio.

Cause Analysis

The last data block in the log file is incorrect and the file size is greater than 256k.

Solution

Log in to the host and delete the log files of the latest date from the /var/dlog directory.
