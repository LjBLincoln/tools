Model Management

Model Management

Models are classified into built-in models, custom models, and Caffe models.

  • BuiltIn Models

    Built-in models are preset in Mind Studio and exist before a workspace is created. You can use them but cannot add, delete, or modify them.

    The currently available built-in model is ResNet-18.

  • My Models

    A custom model (such as a Caffe or TensorFlow model) can be added to Mind Studio by using the offline conversion function for future use. A newly created workspace has no custom models. You can add a custom model by model conversion or simply adding one.

  • Caffe Models

    After a Caffe model is added on the orchestration window, the Caffe model is added to Model Zoo. A newly created workspace has no Caffe models.
