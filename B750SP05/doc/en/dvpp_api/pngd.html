PNGD

PNGD

The PNGD module is used to hardware decode images in PNG format.

  • The PNGD supports the following input formats:

    RGBA and RGB

  • The PNGD supports the following output formats:

    RGBA and RGB

  • The PNGD supports the following resolutions:

    Maximum resolution: 4096 x 4096; minimum resolution: 32 x 32

  • Memory limit:

    • The start address of the cache must be 128-byte aligned.
    • The virtual addresses of the input and output caches for the same task should be in the same 4 GB space.

The performance specifications of the PNGD are as follows:

Input Parameter: struct PngInputInfoAPI

Member Variable

Description

void* inputData

Input image data

unsigned long inputSize

Input buffer length, which is used to verify the input data

void* address

Buffer address, which is internally allocated and does not need to be configured by users

unsigned long size

Buffer size, which does not need to be configured by users

int transformFlag

Conversion flag. The value 1 indicates that RGBA is converted into RGB, while the value 0 indicates that the original format is retained.

Output Parameter: struct PngOutputInfoAPI

Member Variable

Description

void* outputData

Output image data

unsigned long outputSize

Output buffer length

void* address

Output buffer address, which needs to be manually released by users

unsigned long size

Buffer size

int format

Output image format

  • 2: RGB output
  • 6: RGBA output

unsigned int width

Output image width

unsigned int high

Output image height

unsigned int widthAlign

Width alignment. The size of the buffer occupied by each line of data is 128-byte aligned. If the output format is RGB, the result of Width x 3 is 128-byte aligned. If the output format is RGBA, the result of Width x 4 is 128-byte aligned.

unsigned int highAlign

Height alignment. Currently, 16-byte alignment is used.

Calling Example

int test() // Basic test case of the PNGD
{
 // load test file
 FILE *fpIn = fopen("dvpp_png_decode_001", "rb");
 if(NULL == fpIn){
  printf("can not open file dvpp_png_decode_001\n");
  return -1;
 }
 fseek(fpIn, 0, SEEK_END);
 uint32_t fileLen = ftell(fpIn);
 fseek(fpIn, 0, SEEK_SET);
 void* inbuf = malloc(fileLen);
 if(NULL == inbuf){
  printf("can not alloc memmory\n");
  fclose(fpIn);
  return -1;
 }else{
     fread(inbuf, 1, fileLen, fpIn);
 }
 fclose(fpIn);
 // prepare msg
        struct PngInputInfoAPI inputInfo; // Input data
        inputInfo.inputData = inbuf; // Input PNG data
        inputInfo.inoutSize = fileLen; // Length of the input PNG data
       inputInfo.transformFlag = 0; // Whether to convert the format
        struct PngOutputInfoAPI outputInfo; // Output data
       dvppapi_ctl_msg dvppApiCtlMsg; //: MSG for the calling API
    dvppApiCtlMsg.in       = (void*)(&inputInfo);
    dvppApiCtlMsg.in_size  = sizeof(struct PngInputInfoAPI);
    dvppApiCtlMsg.out      = (void*)(&outputInfo);
    dvppApiCtlMsg.out_size = sizeof(struct PngOutputInfoAPI);
    // use interface
    IDVPPAPI *pidvppapi = NULL;
       CreateDvppApi(pidvppapi); // Crearte a DVPP API object.
        if(pidvppapi){// Call the DvppCtl interface to perform DVPP_CTL_PNGD_PROC.
    if(0 != DvppCtl(pidvppapi, DVPP_CTL_PNGD_PROC, &dvppApiCtlMsg)){
        printf("call dvppctl process failed\n");
                DestroyDvppApi(pidvppapi); // Destroy the DVPP API object.
        return -1;
    }
    }else{
        printf("can not get dvpp api\n");
    }
    free(inbuf);
    munmap(outputInfo.address, ALIGN_UP(outputInfo.size, MAP_2M));
    DestroyDvppApi(pidvppapi);
}
