rawdata_overlay_config

rawdata_overlay_config

Member Variable

Description

Value Range

int: image_type

Image type. For either a picture-based or text-based image, the image type must be yuv420_semi_planar.

0

(yuv420_semi_planar)

int: image_rank_type

Image format (NV12 or NV21)

The value is 0 or 1.

0 corresponds to NV12, while 1 corresponds to NV21.

int: bit_width

Bit depth

The value must be 8.

int: in_width_image

Image width

The value must be an even number in the range of [16, 4096].

int: in_high_image

Image height

The value must be an even number in the range of [16, 4096].

int: in_width_text

Width of a text-based image. The width must be smaller than that of a picture-based image.

The value must be an even number in the range of [16, 4096].

int: in_high_text

Height of a text-based image. The width must be smaller than that of a picture-based image.

The value must be an even number in the range of [16, 4096].

int: image_width_stride

Width stride of a picture-based image

Stride of the image data in the buffer. If the image size is not aligned in the buffer, the width stride of a picture-based image is the same as the width of the picture-based image. Otherwise, set this parameter to a value based on an aligned image size.

int: image_high_stride

Height stride of a picture-based image

Stride of the image data in the buffer. If the image size is not aligned in the buffer, the height stride of a picture-based image is the same as the height of the picture-based image. Otherwise, set this parameter to a value based on an aligned image size.

int: text_width_stride

Width stride of a text-based image

Stride of the image data in the buffer. If the image size is not aligned in the buffer, the width stride of a text-based image is the same as the width of the text-based image. Otherwise, set this parameter to a value based on an aligned image size.

int: text_high_stride

Height stride of a text-based image

Stride of the image data in the buffer. If the image size is not aligned in the buffer, the height stride of a text-based image is the same as the height of the text-based image. Otherwise, set this parameter to a value based on an aligned image size.

char *: in_buffer_image

Input buffer of a picture-based image

Ensure that the size of the input buffer of a picture-based image is the same as its width and height after aligned by image_width_stride and image_high_stride.

For a picture-based image in YUV420 semi-planar NV12 format, the image size is (Width after alignment) x (Height after alignment) x 1.5. If the input buffer size is inconsistent with this calculation result, the VPC fails to be invoked.

char *: in_buffer_text

Input buffer of a text-based image

Ensure that the size of the input buffer of a text-based image is the same as its width and height after aligned by text_width_stride and text_high_stride.

For a text-based image in YUV420 semi-planar NV12 format, the image size is (Width after alignment) x (Height after alignment) x 1.5. If the input buffer size is inconsistent with this calculation result, the VPC fails to be invoked.

shared_ptr<AutoBuffer> auto_overlay_out_buffer

Output buffer after overlay

This parameter is a smart pointer transferred into the VPC, used for the caller to allocate the output buffer.
