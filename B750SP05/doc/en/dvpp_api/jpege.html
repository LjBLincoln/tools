JPEGE

JPEGE

The JPEGE module is used to generate a .jpg image based on encoding.

  • The JPEGE supports the following input formats:

    • YUV422Packed (YUYV, YVYU, UYVY, VYUY)
    • YUV420 semi-planar (NV12, NV21)

  • The JPEGE supports the following output format:

    jpeg

  • Memory limit:

    • The start address of the cache must be 128-byte aligned.
    • The virtual addresses of the input and output caches for the same task should be in the same 4 GB space.

The performance specifications of the JPEGE are as follows:

Scene

Total Frame Rate

1080p ≥ 1 channel (no limit on the number of channels)

64 fps

4K ≥ 1 channel (no limit on the number of channels)

16 fps

Input Parameter: sJpegeIn

Member Variable

Description

eEncodeFormat format

Format of the input YUV data. YUV422Packed (YUYV, YVYU, UYVY, VYUY) and YUV420 semi-planar (NV12, NV21) are supported.

typedef enum {

JPGENC_FORMAT_UYVY = 0x0,

JPGENC_FORMAT_VYUY = 0x1,

JPGENC_FORMAT_YVYU = 0x2,

JPGENC_FORMAT_YUYV = 0x3,

JPGENC_FORMAT_NV12 = 0x10,

JPGENC_FORMAT_NV21 = 0x11,

} eEncodeFormat;

unsigned char* buf

YUV input data, which needs to be allocated by the caller and must be properly aligned. For details, see the parameters stride and heightAligned.

unsigned long bufSize

Input buffer length, that is, the data is aligned to PAGE_SIZE(4k).

unsigned int width

Width of the input image. The value range is [32, 8192].

unsigned int height

Height of the input image. The value range is [32, 8192].

unsigned int stride

Width of the input image after alignment. The width is 16-pixel aligned or can be aligned by a multiple of 16, for example, 128. For YUV422Packed data, the stride must be twice that of width, which is also 16-pixel aligned.

unsigned int heightAligned

Height of the input image after alignment. The height can be either 16-pixel aligned or not 16-pixel aligned.

unsigned int level

The encoding quality range is (0, 100]. A smaller value indicates poorer image quality.

Output Parameter: sJpegeOut

Member Variable

Description

unsigned char* jpg_data

Start address of the JPG data in the output buffer

unsigned int jpgSize

Length of the encoded JPG image data

JpegCalBackFree

Callback function for releasing the output buffer

Calling Example

void TEST_5() // jpeg encoder base function
{
    sJpegeIn  inData;
    sJpegeOut outData;

    inData.width         = gWidth;
    inData.height        = gHigh;
    inData.heightAligned = gHigh;
    inData.format        = (eEncodeFormat)gFormat;
    inData.level         = 100;

    if (JPGENC_FORMAT_YUV420 == (inData.format&JPGENC_FORMAT_BIT)) {
        inData.stride  = ALIGN_UP( inData.width, 16 );
        inData.bufSize = inData.stride * inData.heightAligned * 3 / 2;
    } else {
        inData.stride  = ALIGN_UP( inData.width * 2, 16 );
        inData.bufSize = inData.stride * inData.heightAligned;
    }

    void* addrOrig = HIAI_DMalloc(inData.bufSize + 128, MALLOC_DEFAULT_TIME_OUT, HIAI_MEMORY_HUGE_PAGE);

    if (addrOrig == MAP_FAILED) {
        printf("TEST_5, can not alloc input buffer\n");
        return;
    }
    inData.buf = (unsigned char*)ALIGN_UP( (uint64_t)addrOrig, 128 );

    do{
        int err = 0;
        // load img file 
        FILE* fpIn = fopen( in_file_name, "rb" );
        if (fpIn == NULL) {
            printf("TEST_5, can not open input file\n");
            break;                
        }

        if (JPGENC_FORMAT_YUV420 == (inData.format&JPGENC_FORMAT_BIT) ) {   
            // for yuv422packed format, like uyvy / vyuy / yuyv / yvyu
            printf("TEST_5, input yuv 420 data\n"); 
            // for y data
            for(uint32_t j=0; j<inData.height; j++ ) {
                fread( inData.buf + j*inData.stride, 1, inData.width, fpIn );
            }

            // for uv data
            for(uint32_t j = inData.heightAligned; j < inData.heightAligned + inData.height / 2; j++ ) {
                fread( inData.buf+ j*inData.stride, 1, inData.width, fpIn );
            }

        } else {   
            // for yuv420semi-planar format, like nv12 / nv21
            printf("TEST_5, input yuv 422 data\n"); 
            for(uint32_t j = 0; j < inData.height; j++) {
                fread(inData.buf + j * inData.stride, 1, inData.width * 2, fpIn );
            }

        }
        fclose( fpIn );
        if (err == -1) {
            break;
        }

        // call jpege process
        dvppapi_ctl_msg dvppApiCtlMsg;
        dvppApiCtlMsg.in = (void *)&inData;
        dvppApiCtlMsg.in_size = sizeof(inData);
        dvppApiCtlMsg.out = (void *)&outData;
        dvppApiCtlMsg.out_size = sizeof(outData);

        IDVPPAPI *pidvppapi = NULL;
        CreateDvppApi(pidvppapi);
        if (pidvppapi == NULL) {
            printf("TEST_5, can not open dvppapi engine\n");
            break;
        }

        JpegeProcessBranch(pidvppapi, dvppApiCtlMsg, outData);

        DestroyDvppApi(pidvppapi);

    } while(0); // for resource inData.buf
    if(addrOrig != NULL) {
        munmap( addrOrig, ALIGN_UP( inData.bufSize+128 , MAP_2M ) );
    }

}
