Structures in IMAGE_CONFIG

Structures in IMAGE_CONFIG

  • ROI_CONFIG structures

Member Variable

Description

Value Range

CROP_CONFIG: crop_config

Structure of cropping parameter configuration

For details, see CROP_CONFIG.

YUV_SUM_OUT_CONFIG: sum_out

Structure 1 of output parameter configuration

For details, see YUV_SUM_OUT_CONFIG.

NORMAL_OUT_CONFIG: normal_out

Structure 2 of output parameter configuration (reserved, not supported yet)

For details, see NORMAL_OUT_CONFIG.

ROI_CONFIG*: next

Pointer to the next ROI_CONFIG structure

Set this parameter when multiple ROIs are used. Otherwise, set this parameter to NULL.
  • CROP_CONFIG structures

Member Variable

Description

Value Range

int: enable

Whether to perform cropping

0: disabled

1: enabled

unsigned int: hmin

Minimum offset in the horizontal direction

The value is an even number and less than hmax.

unsigned int: hmax

Maximum offset in the horizontal direction

The value is an odd number and less than the input width (in_width).

unsigned int: vmin

Minimum offset in the vertical direction

The value is an even number and less than vmax.

unsigned int: vmax

Maximum offset in the vertical direction

The value is an odd number and less than the input height (in_height).
  • YUV_SUM_OUT_CONFIG structures

Member Variable

Description

Value Range

int: enable

Whether to enable the channel for output

0: disabled

1: enabled

unsigned int: out_width

Output image width

The value is an even number in the range of 128–4096. The scaling coefficient must be in the range of [0.03125, 4].

unsigned int: out_height

Output image height

The value is an even number in the range of 16–4096. The scaling coefficient must be in the range of [0.03125, 4].

char*: out_buffer

Pointer to the address of the output image buffer

The output buffer size must be calculated based on the output image resolution with the 128-pixel aligned width and 16-pixel aligned height.

unsigned int: yuv_sum

Sum of all YUV values of the output image

Reserved (not supported yet)
