JPEGE Instructions

JPEGE Instructions

./sample_dvpp_hlt_ddk --in_image_file dvppapi_jpegd_w1280_h720_YUV420 --test_type 5  --img_width 1280  --img_height 720 --in_format 17

Description:

  • In this sample, a 1280 x 720 YUV image is encoded into a .jpg image.
  • No output file needs to be specified for the JPEGE.
  • The encoding file and result are stored in the current directory.
