DestroyDvppApi

DestroyDvppApi

Prototype

int DestroyDvppApi(IDVPPAPI *&pIDVPPAPI)

Function

Releases the DVPP API and disables the DVPP executor.

NOTE:

Once the DestroyDvppApi function is called, if you want to continue to call the DVPP, you need to obtain the DVPP API instance again.

Input Description

The IDVPPAPI pointer reference is input.

Output Description

There is no output.

Return Value

The return value 0 indicates success, while –1 indicates failure.

Remarks

None

Restrictions

This function is one of the unified calling APIs provided by the DVPP externally and is used to release the DVPP execution resources.

Calling Example

DestroyDvppApi(pidvppapi);
