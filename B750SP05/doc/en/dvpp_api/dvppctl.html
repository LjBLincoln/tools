DvppCtl

DvppCtl

Prototype

int DvppCtl(IDVPPAPI *&pIDVPPAPI, int CMD, dvppapi_ctl_msg *MSG)

Function

Controls the DVPP executor to perform operations such as video encoding and decoding, VPC, JPEG encoding and decoding, and PNG decoding.

Input Description

The input consists of the IDVPPAPI pointer reference, control command word (CMD), and DVPP executor configuration information (MSG) whose type is dvppapi_ctl_msg. The configuration information of each DVPP component is different, and it is uniformly used by the caller in the header file dvpp_config.h. The corresponding parameters must be set during the invocation of functions.

Output Description

The output consists of the output buffer in the configuration information (MSG) and the output status information of the DVPP (which is all stored in the MSG). There may be one or more output buffers, depending on the input parameters of each function. For details, see DvppCtl.

Return Value

The return value 0 indicates success, while –1 indicates failure.

Remarks

The caller calls the DvppCtl function to transfer the IDVPPAPI object pointer, configures dvppapi_ctl_msg of the corresponding function, and transfers the correct control command word (CMD).

Restrictions

This function is one of the unified calling APIs provided by the DVPP externally and is used to call specific functions of the DVPP executor.
