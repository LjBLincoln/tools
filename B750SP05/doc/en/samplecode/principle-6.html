Principle

Principle

The network connectivity application runs on the Atlas 200 DK or the AI acceleration cloud server and implements the inference function by using a common classification network or Faster R-CNN object detection network.

  • For a classification network, the first n inference results are output.
  • For a detection network, the detected objects are marked in the images and the images are saved in .jpg format.
