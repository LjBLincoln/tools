Preparing the Server Environment

The server serves as the installation environment of Mind Studio. The following requirements on the hardware and operating system （OS） must be met.

Table 1 Ubuntu system version information

Category

Version Restriction

Obtaining Method

Precautions

Hardware

≥ 4 GB memory is recommended for a single-user scenario.

-
  • If the memory size of the Linux host is 4 GB, the size of the model file should be less than or equal to 350 MB for model conversion and simulation on Mind Studio. Otherwise, the OS may be unstable.
  • If the configurations of the Linux host are upgraded, for example, the memory is expanded to 8 GB, the supported operation specifications should be increased proportionally.

    For example, if the memory is expanded from 4 GB to 8 GB, the size threshold of the model file should be 700 MB.

OS

16.04.3

Download the required version from http://old-releases.ubuntu.com/releases/16.04.3/. You can download the desktop version ubuntu-XXX-desktop-amd64.iso or server version ubuntu-XXX-server-amd64.iso.

NOTE:

In ubuntu-XXX-desktop-amd64.iso and ubuntu-XXX-server-amd64.iso, XXX indicates the version number.
  • Ensure that the OS supports the UTF-8 character set. Otherwise, you cannot view the support documents by using online help of Mind Studio after the installation. For details about the troubleshooting solution, see What Do I Do If I Cannot View the Online Help Documents?.
  • During model conversion, if a custom TE plug-in is loaded, the GUI may be stuck. Run the uname -r or cat /proc/version command to check whether the Linux kernel version is earlier than 4.18. If so, upgrade the Linux kernel by referring to:

    https://bugzilla.kernel.org/attachment.cgi?id=277305
