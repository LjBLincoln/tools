Software Package Preparation

Before installing the tool, prepare the following software packages:

  • Obtain the software packages from ascenddk-private in https://github.com/Ascend.

Table 1 describes the software packages.

Table 1 Overview of the software packages

Installation Package

Content

Integrity Verification File for the Software Package

Application Scenario

  • install.sh
  • check_sha.sh

mini_mind_studio_developerkit-x.x.x.x.rar.asc

Installation package on Ubuntu, for developer board scenarios

NOTE:

  • x indicates the version number of the software package.
  • Mind-Studio_Ubuntu-x86-64.tar is the Mind Studio installation package. A DDK is distinguished by ddk in its file name. install.sh is used for installation, and check_sha.sh is used to verify the integrity of the preceding three files. （check_sha.sh is automatically called to perform integrity check during the execution of install.sh.）, the add_sudo.sh script is used to add rights to common users, and the del_sudo.sh script is used to delete common user rights.
